# MCSkinn Editor

**A brand new Minecraft Skin Editor for Windows.**

## Screenshot

![](https://github.com/NotYoojun/MCSkinn/blob/main/docs/images/screenshot.png?raw=true)

## Features

### Available Functions

- Skin editing
- Skin library
- Undo/Redo
- All paint tools
- 3D/2D View
- More models

### We're still working on

- Undo/Redo list
- Options menu
- Keyboard Shortcuts
- ...
